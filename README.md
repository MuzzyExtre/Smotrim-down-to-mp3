# Парсер подкастов с автоматическим скачиванием и отправкой в Telegram

## Описание
Скрипт автоматически парсит выпуски подкаста "Провокация" с сайта smotrim.ru, скачивает аудиофайлы, сжимает их при необходимости и отправляет в Telegram-канал.

## Основные возможности
- Парсинг сайта и извлечение новых выпусков
- Скачивание и сжатие аудиофайлов
- Отправка аудио в Telegram-канал
- Очистка временных файлов
- Планировщик задач

## Установка
1. Установите Python 3.8+
2. Установите ffmpeg и добавьте его в PATH
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Запуск
1. Настройте config.ini (пример в ТЗ)
2. Запустите основной скрипт:
   ```bash
   python main.py
   ```
