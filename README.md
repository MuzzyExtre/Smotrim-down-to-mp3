## `$ cat features.txt`

```

│  ▸ Парсинг выпусков через API smotrim.ru             │
│  ▸ Скачивание MP3 с автоповтором при сбоях           │

│  ▸ Цветной вывод прогресса в терминал                │
│  ▸ Гибкая конфигурация через config.ini              │
└──────────────────────────────────────────────────────┘
```

<br>

## `$ ./install.sh`

> **Требования:** Python 3.8+ · FFmpeg · Telegram Bot Token

```bash
# ── 01 Клонировать репозиторий ──
git clone https://github.com/MuzzyExtre/Smotrim-down-to-mp3.git
cd Smotrim-down-to-mp3

# ── 02 Установить зависимости ──
pip install -r requirements.txt

# ── 03 Создать конфигурацию ──
cp config.ini.example config.ini
```

<br>

## `$ nano config.ini`

```ini
[telegram]
# Токен бота (получите у @BotFather)
bot_token = "ВАШ_ТОКЕН"
# Канал для отправки подкастов
channel_id = "@ВАШ_КАНАЛ"

[audio]
# Лимит размера файла (МБ)
max_file_size_mb = 50
# Качество сжатия: low | medium | high
compression_quality = medium

[ffmpeg]
# Оставьте пустым для автопоиска в PATH
ffmpeg_path = ""
ffprobe_path = ""

[logging]
log_level = INFO
save_logs = true
```

## `$ cat dependencies.json`

| Пакет | Версия | Назначение |
|:------|:------:|:-----------|
| `requests` | ≥ 2.25 | HTTP-запросы, скачивание файлов |
| `beautifulsoup4` | ≥ 4.9 | Парсинг HTML |
| `ffmpeg-python` | ≥ 0.2 | Сжатие аудио |
| `colorama` | ≥ 0.4 | Цветной вывод в терминал |
| `python-telegram-bot` | ≥ 13.0 | Telegram Bot API |
| `pydub` | ≥ 0.25 | Обработка аудио |
| `mutagen` | ≥ 1.45 | Метаданные аудиофайлов |
| `schedule` | ≥ 1.1 | Планировщик задач |
